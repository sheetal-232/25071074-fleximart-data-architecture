# FlexiMart Data Architecture Project

## Project Overview
This project demonstrates an end-to-end data architecture workflow including ETL processing, relational databases, NoSQL analysis, and data warehousing for an e-commerce platform.

## Project Structure
- data/ : Raw input CSV files
- part1-database-etl/ : ETL pipeline, schema documentation, SQL queries
- part2-nosql/ : MongoDB analysis and operations
- part3-datawarehouse/ : Star schema, warehouse data, analytics queries
